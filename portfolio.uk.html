📋 Проєкти

Це тільки частина проєктів. Натисніть на кнопку, щоб побачити деталі

Day Mode

Додаток для відстеження власних біоритмів і стану. Реалізовано запис і відтворення звуку, додавання нотаток, локальні push-сповіщення. Кореляція зібраних даних розраховується на бекенді.

Стек: Flutter Riverpod SQLite SharedPreferences auto_route flutter_secure_storage dio flutter_local_notifications Python FastAPI

Темна і світла теми. Локальний бекенд на Python з синхронізацією через FastAPI.

TradeSizer

Офлайн калькулятор для розрахунку позицій і стоп-лосу для криптовалютної біржі. Впроваджено нативне «Поділитися додатком», оплата Google, Firebase Analytics і Crashlytics, push-сповіщення і Remote Config.

Стек: Flutter Firebase FCM Push Remote Config Crashlytics Google Play Billing

Реалізовано ознайомлення з додатком, генерація ключів і підпису, реєстрація акаунту в Google Play, публікація і тестування. Анімація в Blender.

🎬 Дивитися рекламне відео для Google Play

Pull Sky

Магазин з продажу натяжних стель. Участь у розробці, допомога в добудові проєкту.

Стек: Flutter Cubit SharedPreferences REST API Figma

Калькулятор

Пробний, один з перших проєктів. Підключення стороннього API погоди, робота з базою даних.

Стек: Flutter Cubit/BLoC SharedPreferences Weather API SQLite

Локальна реєстрація і авторизація користувача.

Rick and Morty

Додаток з інформацією про персонажів із серіалу «Рік і Морті». Завантаження списків, кешування даних, реалізовано пошук.

Стек: Flutter BLoC Rick and Morty API Dio Cache
